# web 前端面试题

该文档整理记录了一些前端的面试题。

本书将以面试的场景来阐述前端的相关技术。

适合具有一定web基础的读者阅读。

本书将分为以下内容：

  * CSS 相关

  * HTML 相关

  * 网络 相关

  * javaScript 相关

  * React 相关

  * Vue  相关

  * webpack 相关

  * 性能优化与web工程 相关

  * 算法 相关

## 前提

  假设对javaScript CSS HTML有一定的了解。
  
  并且对babel也有一定的了解。

  对webpack、node、npm(yarn)等也有一些了解。
